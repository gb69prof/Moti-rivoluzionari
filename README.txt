## Come aprire
Apri `pages/index.html` con un browser (Chrome/Safari/Edge).
Se la apri da file locale e il browser blocca qualche risorsa, usa un piccolo server locale:

- Python:
  python3 -m http.server 8000

Poi vai su: http://localhost:8000/pages/

## Struttura
- pages/ : pagine HTML (Home + 4 temi)
- assets/ : CSS/JS + schemi (SVG) + schema generale (PNG)
